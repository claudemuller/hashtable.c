# Hash Table

A naive hash table implementation written in C.

# Requirements

- [gcc](https://gcc.gnu.org/)
- [make](https://www.gnu.org/software/make/)

# Build

```bash
make build
```

# Run

```bash
make run
```

# Build a Debug Binary

```bash
make debug-build
```

# Start `lldb` or `gdb` With Debug Binary

```bash
make debug
```
